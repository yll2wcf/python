#定义函数
在Python中，定义一个函数要使用`def`语句，依次写出函数名、括号、括号中的参数和冒号`:`，然后，在缩进块中编写函数体，函数的返回值用return语句返回。
```py
def my_abs(x):
    if x >= 0:
        return x
    else:
        return -x
```

##空函数
如果想定义一个什么事也不做的空函数，可以用`pass`语句：
```py
def nop():
    pass
```

>`pass`语句什么都不做，那有什么用？

实际上`pass`可以用来作为占位符，比如现在还没想好怎么写函数的代码，就可以先放一个`pass`，让代码能运行起来。

`pass`还可以用在其他语句里，比如：
```py
if age >= 18:
    pass
```
缺少了`pass`，代码运行就会有语法错误。
